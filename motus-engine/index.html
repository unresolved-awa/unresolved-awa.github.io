Motus Engine

a 3D game engine built with C++ & Slint

about

Motus Engine is a work-in-progress 3D game engine written in C++23 with a Slint-based editor. The engine features a custom windowing layer, full editor UI with outliner, inspector, viewport, and file system panels.

features

editor UI

outliner, inspector, 3D viewport, and file system panels with dark theme

cross-platform ready

built on Slint UI framework, platform abstraction for future Linux & macOS
